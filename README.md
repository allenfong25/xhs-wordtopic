## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and fill in your API key:
   ```bash
   cp .env.example .env.local
   ```
   Then edit `.env.local` to set `GEMINI_API_KEY=your_actual_api_key`
3. Run the app:
   `npm run dev`
